# PanenHUB APP - Prototype

**IMPORTANT:**
This repository contains a static HTML/CSS prototype of the PanenHUB application. It is **only a UI/UX prototype** for demonstration and conceptual purposes. The application does not contain a real working backend, database, or API integrations. All data shown is mock data.

## Project Overview

PanenHUB is an application concept aimed at connecting farmers and helping them manage their harvests, distributions, marketplace transactions, subsidies, and fulfillment tracking.

## How to View
To interact with the prototype, you do not need any local server setup. Simply open the `login.html` file in your preferred web browser to begin the application flow. 

The links and buttons have been configured to navigate you intuitively through the main screens:
- `login.html` & `registrasi_panenhub_simpel.html` - Auth flows
- `dashboard.html` - The central hub after login
- `marketplace_history.html` & `ambil_request_komoditas.html` - Marketplace and orders
- `validasi_pengiriman.html` - Scanner mockups
- Additional pages for profile (`profile_settings.html`), subsidy info (`informasi_subsidi.html`), etc.

## Technology Stack
Built strictly using:
- **HTML5**
- **Tailwind CSS** (via CDN plugin for quick prototyping)
- **Google Fonts** (Inter and Material Symbols)
